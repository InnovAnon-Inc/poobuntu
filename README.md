# poobuntu
==========

Parallelization-hack-enabled Ubuntu

> "Poobuntu! It's fast as sh*t"
> -- nobody

# Usage
`./run.sh`

`./run.sh 18.04`

`./run.sh 16.04`

# Cleaning up all the poo in your child Dockerfile
WORKDIR /

RUN ./poobuntu-clean.sh

RUN rm -v poobuntu-clean.sh
